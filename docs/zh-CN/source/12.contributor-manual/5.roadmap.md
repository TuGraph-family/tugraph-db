# 技术规划

## 1. 简介

该文档是 TuGraph-DB 未来开发的规划，包括正在开发中、不在开发计划里、已经开发完成但不在开源版本里的功能等。

TuGraph-DB定位开源高性能图数据库，图数据采用集中存储的方式，短期 **不会** 考虑基于分片（Sharding）的数据切分，
而将支持主备复制的模式来解决高并发读的场景，采用云上存储解决存储容量的问题。
